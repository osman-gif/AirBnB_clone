 Project Description

 This project include a console project, This interpreter (the console)
 will act as a shell for the project but limited to specific use case.
 Use it to manage our project's object:

 >> Create a new object (ex: a new User or a new Place)
 >> Retrieve an object from a file, a database etc…
 >> Do operations on objects (count, compute stats, etc…)
 >> Update attributes of an object
 >> Destroy an object

 Usage of the command intepreter:
   	To starting the intepreter type 'console.py', you will get this prompt: (hbnb)
	To get help, type 'help'.
	To exit or quit type 'quit'
	If you typed nothing you get all possiple commands.
	To create an object, type 'create object_name'.
	To destroy an object type 'destroy object_name'.
